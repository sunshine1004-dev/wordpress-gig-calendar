== Description ==

Show your Optune gigs on your WP website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wordpress-gig-calendar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Optune Calendar screen to configure the plugin: specify your Optune username.
4. Use shortcode - [gig-calendar] to display calendar where you want.
